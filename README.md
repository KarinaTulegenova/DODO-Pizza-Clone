

# DODO Pizza Clone

A Pizza Delivery website built with Next.js and TypeScript.
<img width="1180" height="862" alt="image" src="https://github.com/user-attachments/assets/454b6ee2-e098-40a1-ba5e-b01069b09c24" />

Quick facts
- Framework: Next.js 15 (app router)
- Language: TypeScript
- DB tooling: Prisma
- UI libs: Radix UI, lucide-react, zustand, Tailwind-related packages
- Scripts available (exact): npm run dev, npm run build, npm run start, npm run lint, npm run prisma:push, npm run prisma:studio, npm run prisma:seed
